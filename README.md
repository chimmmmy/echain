# echain

a smart keychain with an e ink display, powered by seed studio nrf52840

# objectives

1. show a smart pet on the front that can be interacted with via the two buttons on the back of the device

2. show music and notifications from phone by using bluetooth

3. off screen should show a customizable image that can be sent to the device over bluetooth

4. battery should last at LEAST a month

5. utilize the nrf52480's deep sleep function to increase battery life

6. prototypes out of plastic, final out of cnc aluminum



# current capabilities (untested)

* show a basic landscape display

* connect to an android phone via bluetooth

* change off screen via bluetooth

* show music currently being played via bluetooth

* interactable via 2 buttons

* able to enter deep sleep and exit deep sleep
